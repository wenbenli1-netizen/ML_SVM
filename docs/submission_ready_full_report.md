# 基于 Lecture_Note 的脑 MRI 支持向量机建模报告（整合提交版）

## 1. 研究题目

**基于软间隔支持向量机的脑 MRI 影像 AD 与 NORMAL 分类研究**

## 2. 研究背景

阿尔茨海默病（AD）的早期识别具有重要临床意义。脑 MRI 影像能够反映脑结构变化，因此适合作为疾病辅助识别的数据基础。支持向量机（SVM）是经典的监督学习方法，尤其适用于中小样本、高维特征建模场景。结合当前课程 `Lecture_Note` 中关于“间隔、对偶问题、核函数、软间隔与正则化”的内容，选择 MRI 影像的 SVM 分类任务作为课程建模项目，具有较强的理论对应性与实践可行性。

## 3. 数据分析

### 3.1 数据格式

本项目数据位于 `processed/` 目录，文件格式为 `.nii.gz`，属于 NIfTI 医学影像格式，适合存储三维脑 MRI 数据。

文件命名结构为：

`数据源 + 受试者编号 + 类别标签 + 时间点 + 场强`

例如：

`ADNIADsim+002_S_0619+AD-m00-1.5T.nii.gz`

## 4. 选题与建模方案

### 4.1 与讲义内容的对应

本项目与讲义内容的对应关系如下：

1. `6.1 间隔与支持向量`：解释最大间隔分类思想；
2. `6.2 对偶问题与 KKT`：说明 SVM 由支持向量决定的理论基础；
3. `6.3 核函数`：比较线性核与 RBF 核在 MRI 分类中的差异；
4. `6.4 软间隔与正则化`：通过参数 `C` 实现间隔与误分类惩罚的平衡。

### 4.2 为什么选 `AD vs NORMAL`

最终将主任务设为 `AD vs NORMAL`，原因如下：

1. 标签边界最清晰；
2. 任务最符合讲义中的二分类 SVM 框架；
3. 分类结果最好，最适合作为课程项目主线；
4. 便于在报告中讨论线性核与核 SVM 的差异。

### 4.3 建模流程

本项目采用如下建模流程：

1. 选取 `m00` 时间点；
2. 从 `.nii.gz` 中读取三维 MRI 体数据；
3. 做强度裁剪与图像内标准化；
4. 下采样到 `32 x 32 x 32`；
5. 展平成向量；
6. 使用 `StandardScaler + PCA(100)` 做特征变换；
7. 比较线性 SVM 与 RBF-SVM；
8. 使用受试者级划分和分组交叉验证评估性能。

## 5. 实验设计

### 5.1 数据划分策略

为避免数据泄漏，本项目按受试者编号切分训练集和测试集，并在训练集内部采用 `StratifiedGroupKFold` 做交叉验证，保证同一受试者不会同时出现在训练和测试中。

### 5.2 模型设置

比较两个模型：

1. Linear SVM
2. RBF-SVM

参数搜索范围为：

1. 线性核：`C ∈ {0.1, 1, 10}`
2. RBF 核：`C ∈ {0.1, 1, 10}`，`gamma ∈ {scale, 0.1, 0.01}`

### 5.3 评估指标

实验使用以下指标：

1. Accuracy
2. Balanced Accuracy
3. Precision
4. Recall
5. F1-score
6. ROC-AUC

## 6. 实验结果

### 6.1 主实验：AD vs NORMAL

| 模型 | 最优参数 | Accuracy | Balanced Accuracy | F1 | ROC-AUC |
| --- | --- | ---: | ---: | ---: | ---: |
| Linear SVM | `C=0.1` | 0.7692 | 0.7721 | 0.7500 | 0.9071 |
| RBF-SVM | `C=10, gamma=scale` | 0.7564 | 0.7507 | 0.7164 | 0.8730 |

可以看到，在当前 PCA 特征表示下，线性 SVM 的整体表现优于 RBF-SVM。特别是 ROC-AUC 达到 `0.9071`，说明 `AD` 与 `NORMAL` 的可分性较强。

## 7. 结果分析

### 7.1 主任务难度最低

四组任务中，`AD vs NORMAL` 的分类结果最好，因此最适合作为提交报告的主任务。

### 7.2 RBF 核并非始终优于线性核

当前实验结果显示：

1. 在 `AD vs NORMAL` 中，线性核表现更好；
2. 在 `MCI vs NORMAL` 中，RBF 核略优；
3. 在三分类中，线性核略稳。

这说明核函数不是越复杂越好，其表现取决于数据分布和特征表示。

### 7.3 扩展任务更难

`MCI` 与 `AD` 或 `NORMAL` 之间的边界更加复杂，因此相关任务结果明显差于 `AD vs NORMAL`。三分类任务最难，说明在更复杂问题中需要更强的特征工程。

## 8. 结论

本文基于 `Lecture_Note` 中关于 SVM 的理论内容，围绕脑 MRI 数据开展了分类建模研究。首先通过数据分析确认该数据集具备统一建模条件；随后在 `m00` 基线时间点上构建了多个 SVM 分类任务。实验结果表明，`AD vs NORMAL` 是最适合课程报告主线的任务，其中线性 SVM 取得了最佳表现（`Accuracy=0.7692`，`Balanced Accuracy=0.7721`，`ROC-AUC=0.9071`）。因此，选择“基于软间隔支持向量机的脑 MRI 影像 AD 与 NORMAL 分类研究”作为本课程建模题目是合理且有效的。
