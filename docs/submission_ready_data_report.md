# processed 数据分析报告（提交版）

## 1. 数据来源与分析目的

本报告针对项目目录 `processed/` 中的 MRI 数据进行系统统计与结构分析，目标是回答三个问题：

1. 数据集的标签、时间点和样本规模分别是什么；
2. 这些 `.nii.gz` 文件的空间尺寸、分辨率和数据类型是否一致；
3. 该数据集是否适合直接用于基于 SVM 的脑 MRI 分类建模。

本报告中的数字均来自对当前全部数据文件的实际统计，相关统计脚本见 `scripts/profile_dataset.py`。

## 2. 文件格式与命名规则

数据文件均为 `NIfTI` 格式，即 `.nii.gz` 文件。这是一种医学影像中常用的三维数据存储格式，适合保存脑 MRI 体数据。

文件名统一满足如下结构：

`数据源 + 受试者编号 + 类别标签 + 时间点 + 场强`

例如：

`ADNIADsim+002_S_0619+AD-m00-1.5T.nii.gz`

可以解析为：

1. 数据源：`ADNIADsim`
2. 受试者编号：`002_S_0619`
3. 类别标签：`AD`
4. 时间点：`m00`
5. 扫描场强：`1.5T`

经检查，当前目录中全部文件都能被上述规则解析，无命名异常。

## 3. 数据规模统计

### 3.1 总体规模

`processed/` 目录共包含 `2989` 个影像文件。

按类别统计如下：

| 类别 | 文件数 | 受试者数 |
| --- | ---: | ---: |
| AD | 577 | 172 |
| MCI | 1598 | 380 |
| NORMAL | 814 | 218 |

可以看出，`MCI` 类数量最多，`AD` 类数量最少，数据集整体存在一定类别不均衡现象。

### 3.2 时间点分布

按时间点统计如下：

| 时间点 | 文件数 |
| --- | ---: |
| m00 | 768 |
| m06 | 728 |
| m12 | 677 |
| m18 | 284 |
| m24 | 482 |
| m36 | 50 |

按“类别-时间点”联合统计如下：

| 类别 | m00 | m06 | m12 | m18 | m24 | m36 |
| --- | ---: | ---: | ---: | ---: | ---: | ---: |
| AD | 172 | 163 | 143 | 0 | 99 | 0 |
| MCI | 378 | 354 | 335 | 284 | 222 | 25 |
| NORMAL | 218 | 211 | 199 | 0 | 161 | 25 |

这个分布说明：

1. `m00` 是样本最多且三类都齐全的时间点；
2. `m18` 仅在 `MCI` 中出现；
3. `m36` 仅在 `MCI` 和 `NORMAL` 中出现；
4. 若直接开展纵向建模，需要考虑时间点不完整问题。

### 3.3 完整随访情况

同时拥有 `m00/m06/m12/m24` 四个时间点的受试者数如下：

| 类别 | 完整受试者数 |
| --- | ---: |
| AD | 91 |
| MCI | 202 |
| NORMAL | 152 |

由此可见，数据中已经存在一批可用于纵向分析的完整随访个体，但对于课程项目而言，先使用 `m00` 构建基线分类模型更稳妥。

## 4. 影像头信息分析

对全部 2989 个文件的头信息检查得到如下结论：

| 项目 | 统计结果 |
| --- | --- |
| 图像尺寸 | 全部一致，为 `133 x 129 x 170` |
| 体素间距 | 全部一致，为 `1.0 x 1.0 x 1.0 mm` |
| 数据类型 | 全部一致，为 `int8` |
| 仿射矩阵 | 全部一致 |
| 扫描场强 | 全部为 `1.5T` |

这一结果具有重要意义：

1. 所有 MRI 已被处理到统一空间中；
2. 样本之间不存在分辨率和体积大小不一致的问题；
3. 这批数据非常适合直接进入机器学习建模阶段。

换言之，本数据集不是原始 DICOM 数据，而是已经具备统一分析条件的预处理脑 MRI 数据。

## 5. 基线时间点的像素值统计

以下统计基于全部 `m00` 样本。

| 类别 | m00 样本数 | 平均零值比例 | 平均均值 | 平均标准差 | 最小值下界 | 最大值上界 | 非零体素平均 95 分位 | 非零体素平均 99 分位 |
| --- | ---: | ---: | ---: | ---: | ---: | ---: | ---: | ---: |
| AD | 172 | 0.5617 | -0.9130 | 13.0846 | -122 | 127 | 29.84 | 52.37 |
| MCI | 378 | 0.5620 | -0.7794 | 12.8150 | -118 | 127 | 29.64 | 50.08 |
| NORMAL | 218 | 0.5626 | -0.6113 | 12.4495 | -123 | 127 | 29.21 | 46.90 |

这些数值表明：

1. 平均约 `56%` 的体素值为 `0`，可以理解为影像背景区域占比较大；
2. 数据中同时存在正值与负值，说明其并非未经处理的原始 MRI 灰度，而更可能是经过预处理或标准化后的结果；
3. 三类样本的数值范围接近，整体分布稳定，便于统一特征处理和建模。

## 6. 数据分析结论

综合以上统计，可以得到如下结论：

1. 该数据集标签清晰，包含 `AD`、`MCI` 和 `NORMAL` 三类脑 MRI 数据；
2. 数据同时具有时间序列信息，适合做基线分类与后续纵向分析；
3. 所有样本的空间尺寸、体素间距和仿射矩阵完全一致；
4. 数据已经具备直接开展机器学习建模的条件；
5. 对于课程项目，最推荐的起点是使用 `m00` 构建基线分类任务。

因此，从可操作性、稳定性与课程适配性角度出发，本项目最适合首先开展如下任务：

**在 `m00` 基线时间点上，基于 MRI 影像开展 `AD vs NORMAL` 的 SVM 二分类建模。**

```python
data = img.get_fdata()

print(data.shape)
print(img.get_data_dtype())
print(img.header.get_zooms()[:3])
print(img.affine)
```
