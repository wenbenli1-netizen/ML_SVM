# processed 数据分析报告

## 1. 报告目的

这份报告解决两个最基础的问题：

1. `processed/` 里到底存的是什么数据
2. 这些 `.nii.gz` 文件应该怎么读

报告基于对当前 `processed/` 目录中全部 2989 个文件的统计结果整理而成。

## 2. 数据位置与文件格式

数据目录：

`processed/`

文件格式：

`*.nii.gz`

这是一种常见的医学影像格式，叫 **NIfTI**。它通常用来保存三维或四维脑 MRI、CT、fMRI 等数据。

在 Python 里，最常用的读取库是：

```python
import nibabel as nib
img = nib.load("xxx.nii.gz")
data = img.get_fdata()
```

其中：

1. `img` 是影像对象，包含头信息和空间信息
2. `data` 是真正的三维体素数组

## 3. 文件命名规则

文件名可以统一解析为：

`数据源 + 受试者编号 + 类别标签 + 时间点 + 扫描场强`

例如：

`ADNIADsim+002_S_0619+AD-m00-1.5T.nii.gz`

可拆解为：

1. 数据源：`ADNIADsim`
2. 受试者：`002_S_0619`
3. 标签：`AD`
4. 时间点：`m00`
5. 扫描场强：`1.5T`

目前目录中全部文件都能被同一规则解析，没有异常命名。

## 7. 我们现在可以怎样理解这批数据

从数据形态上看，这是一批：

1. 已经配准到统一空间的三维脑 MRI
2. 带有疾病标签 `AD/MCI/NORMAL`
3. 带有随访时间点 `m00/m06/m12/m24...`
4. 适合做分类，也适合后续做纵向分析

换句话说，这不是“原始 DICOM 扫描文件”，而是一批**已经处理好的机器学习输入数据**。

这也是为什么我们能比较直接地做：

1. `AD vs NORMAL`
2. `MCI vs NORMAL`
3. `AD vs MCI`
4. 三分类 `AD/MCI/NORMAL`

## 8. 你们应该怎么读数据

### 8.1 最小读取方法

下面是最小可用代码：

```python
import nibabel as nib

path = r"processed\\ADNIADsim+002_S_0619+AD-m00-1.5T.nii.gz"
img = nib.load(path)
data = img.get_fdata()

print("shape:", data.shape)
print("dtype:", img.get_data_dtype())
print("zooms:", img.header.get_zooms()[:3])
print("affine:\\n", img.affine)
```

运行后你们会看到：

1. `shape` 是三维数组大小
2. `dtype` 是存储类型
3. `zooms` 是体素间距
4. `affine` 是体素坐标到真实空间坐标的映射

### 8.2 怎么看一张 MRI

MRI 是三维数据，不能像普通图片那样只看一张平面图。通常做法是取中间切片看：

```python
import nibabel as nib
import matplotlib.pyplot as plt

img = nib.load(r"processed\\ADNIADsim+002_S_0619+AD-m00-1.5T.nii.gz")
data = img.get_fdata()

z = data.shape[2] // 2
plt.imshow(data[:, :, z], cmap="gray")
plt.axis("off")
plt.show()
```

## 9. 数据分析对实验设计的意义

### 9.1 为什么先用 `m00`

因为：

1. `m00` 样本最多
2. 所有类别都有 `m00`
3. 作为基线时间点，定义最清楚
4. 可以先避免纵向缺失问题

### 9.2 为什么适合先做 `AD vs NORMAL`

因为：

1. 标签最清楚
2. 任务最容易讲明白
3. 通常可分性最好
4. 最适合用来展示 SVM 的最大间隔和核函数效果

### 9.3 为什么不建议一开始就做纵向模型

因为纵向任务会引入：

1. 时间点不完整
2. 配对样本筛选
3. 变化特征构造
4. 更复杂的实验解释

对于课程项目，先把 `m00` 基线分类做扎实更稳。

## 10. 当前结论

这批 `processed` 数据已经非常适合直接开展 SVM 实验：

1. 文件命名规范，标签可直接解析
2. MRI 空间尺寸完全一致
3. 体素间距一致，仿射一致
4. 数据已具备统一建模条件
5. `m00` 子集非常适合作为第一版实验集

最推荐的起步方式仍然是：

**先做 `m00` 上的 `AD vs NORMAL` 分类，再逐步扩展到 `MCI` 和三分类。**
